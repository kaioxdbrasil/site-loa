Termo de Uso e Consentimento para Coleta de Dados Pessoais

1. Introdução

Este documento tem como objetivo informar os usuários sobre a coleta, uso e proteção dos dados pessoais fornecidos, incluindo nome, e-mail e telefone. Ao fornecer seus dados, você concorda com os termos e condições aqui descritos.

2. Coleta de Dados

Os seguintes dados pessoais serão coletados de forma opcional, ficando a critério do usuário informá-los ou não:

  • Nome
  • E-mail
  • Telefone
  • Município de residência

3. Finalidade da Coleta de Dados

Os dados coletados serão utilizados para as seguintes finalidades:

  • Identificação do usuário
  • Comunicação de informações relevantes relacionadas ao serviço

4. Compartilhamento de Dados

Seus dados pessoais não serão compartilhados com terceiros sem o seu consentimento, exceto quando exigido por lei.

5. Segurança dos Dados

Implementamos medidas de segurança apropriadas para proteger seus dados pessoais contra acesso não autorizado, alteração, divulgação ou destruição.

6. Direitos do Usuário

Você tem o direito de:

  • Acessar seus dados pessoais
  • Corrigir dados incompletos, inexatos ou desatualizados
  • Solicitar a exclusão de seus dados pessoais

Para exercer seus direitos, entre em contato conosco através do e-mail consultapopular@ac.gov.br.

7. Alterações no Termo de Uso

Este termo de uso pode ser atualizado, caso necessário. Qualquer alteração será comunicada através do nosso site ou por e-mail.

8. Consentimento

Ao fornecer seus dados pessoais, você declara ter lido e concordado com os termos e condições aqui apresentados.

Se você tiver dúvidas ou preocupações sobre este termo de uso, entre em contato conosco.
